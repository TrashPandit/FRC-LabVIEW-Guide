How to get a template started

When you first open up labview, you should see this screen

Next you will want to click the "FRC roboRIO Project" template

You will see this screen next

Project name: Enter the name of whatever you want to call the project, usually "'year' Robot Project"

Project folder: Where the LabVIEW project will be saved, usually kept default

Team Number: Enter your team's number here (Votech is 1254)

Keep everything else default and then press the "Finish" button at the bottom

This will open up your LabVIEW, this is all for this tab, go onto the "File Understanding" tab for the next lesson
